# Droping-Droping
Its a simple droping balls game

## steps to Play the Game
  - ##### Install Python

    - https://www.python.org/downloads/

  - ##### Install PYgame
    - ```pip install pygame```

  - ##### Run Python File
    - ``` python Droping-Droping.py ```
